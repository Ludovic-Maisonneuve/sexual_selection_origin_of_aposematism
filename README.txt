
Programs related to the following preprint article : Maisonneuve, L., Aubier, T., The `sexual selection hypothesis' for the origin of aposematism. https://doi.org/XXX

Ludovic Maisonneuve and Thomas Aubier wrote all the scripts with the help of Arthur Weyna.

Content:

- fig_1_to_4: contains Mathematica notebooks to generate figures 1, 2, 3 and 4 of the publication

- fig_5_and_tab_1_&_2: contains python files to generate figure 1 and tables of the publication
